Please see below tables for AuraArtistry API defined endpoints.



Home

HTTP Methods Path Description
GET /(API Home Documentation Page) This is the home page of the Aura Artistry API. It contains information about the API and it's endpoints.



Users

HTTP Methods Path Description
GET /users This endpoint when used retrieves all the registered users and their details from the database.
GET /user/:id This endpoint is used to retrieve a specific product and it's details according to their UserID from the database.
POST /user/register This endpoint is used to register a new user to the database.
POST /user/login This endpoint is used to login a registered user/admin to AuraArtistry.
PATCH/PUT /user/:id This endpoint is used to update a registered user's details according to their UserID from the database.
DELETE /user/:id This endpoint is used to remove a registered user and their details according to their UserID from the database.



Products

HTTP Methods Path Description
GET /products This endpoint when used retrieves all the products and their details from the database.
GET /product/:id This endpoint is used to retrieve a specific product and it's details according to it's prodID from the database.
POST /product/add This endpoint is used to add a new product to the database.
PUT/PATCH /product/:id This endpoint is used to update a product's details according to it's prodID from the database.
DELETE /product/:id This endpoint is used to remove a product and it's details according to it's prodID in the database.
